## Installation

Install tcp server :

```bash
   git clone https://github.com/pawitpr/tcp-speedtest-python

   cd tcp-speedtest-python
   
   python server.py
```

client:
  ```bash
    git clone https://github.com/pawitpr/tcp-speedtest-python

    cd tcp-speedtest-python
     
    <open the client.py and change server ip to your server ip > 

    python client.py
```
